Facial Recognition

Description

The first one is a Facial Recognition App that uses an API to detect faces inside pictures.

To use it, just copy the image address of any photo that you find online into the search bar, press "detect" and it will create a box around the faces.

This project uses HTML5, CSS3, JavaScript, and React.js .

Feline & Company

Description

The second one is a searchable cardlist. It's using a JSON API to create randomized user information. It also uses the Robohash API to create randomized images.

To use it, type in the name of the cat you like, and it will filter through the list to pick out the right one.

This project uses HTML5, CSS3, JavaScript, and React.js .

Tip Calculator

Description

The calculator allows you to rate the food/service you received at the restaurant and calculates the percentage you should give on the tip.

The rating ranges from Outstanding (30%) – Terrible (5%). Also, if you ate with other people, the app also helps you divide how much everyone should contribute to the tip based on how much the food cost, and the total amount of people.

This project uses HTML5, CSS3, and JavaScript.
